# חלק א': סקירה כללית

## 1. מטרת הפרויקט

### 1.1 תיאור הפרויקט

יצירת **אתר אינטראקטיבי מקיף בעברית** המציג את כל מערכות התחבורה ההמונית בעולם - רכבות תחתיות (מטרו), רכבות קלות, חשמליות, ו-BRT - עם דגש על:

- 🗺️ **מפות** - רשמיות, היסטוריות וסכמטיות
- 📊 **נתונים סטטיסטיים** - ק"מ, תחנות, קווים, נוסעים
- 📜 **היסטוריה** - התפתחות המערכות לאורך השנים
- 🎨 **עיצוב** - השראה ממפות תחבורה המוניות

הפרויקט בהשראת הספר "Transit Maps of the World" מאת Mark Ovenden.

### 1.2 ערך מוסף

**למה הפרויקט הזה חשוב?**

1. **משאב ייחודי בעברית** - אין משאב דומה בשפה העברית
2. **חינוכי** - למידה על גיאוגרפיה, היסטוריה, עיצוב מידע
3. **שימושי** - עזרה למטיילים בתכנון נסיעות
4. **קהילתי** - בניית קהילה של חובבי תחבורה ומפות
5. **נגיש** - זמין לכולם, חינם, במכשירים ניידים

### 1.3 מאפיינים טכנולוגיים מרכזיים

- ✅ **PWA (Progressive Web App)** - עבודה במצב offline, התקנה למסך הבית
- ✅ **RTL מלא** - תמיכה מושלמת בעברית מימין לשמאל
- ✅ **Responsive** - עובד מצוין בכל מכשיר
- ✅ **מודולרי** - ניתן להרחבה בשלבים
- ✅ **מהיר** - טעינה מהירה, אנימציות חלקות

## 2. קהל יעד

### 2.1 קהלים ראשיים

**חובבי תחבורה ציבורית** 🚇
- גיל: 25-65
- עניין: מפות, תחבורה, תכנון עירוני
- התנהגות: מחפשים מידע מפורט, משווים מערכות

**מעצבים גרפיים ואדריכלים** 🎨
- גיל: 25-50
- עניין: עיצוב מידע, אדריכלות תחבורתית
- התנהגות: מחפשים השראה, מפות היסטוריות

**תיירים ומטיילים** ✈️
- גיל: 20-60
- עניין: טיול, תרבות, חוויות
- התנהגות: מתכננים ביקורים, צריכים מידע מעשי

**סטודנטים וחוקרים** 📚
- גיל: 18-40
- עניין: מחקר, לימודים, נתונים
- התנהגות: מחפשים מקורות מידע, סטטיסטיקות

### 2.2 פרסונות משתמש

**דני - חובב תחבורה (35)**
- עובד בהייטק, תחביב: מפות ותחבורה
- מטייל הרבה בחו"ל
- רוצה: מידע מפורט, מפות להדפסה, השוואות
- משתמש: Desktop בעבודה, Mobile בנסיעות

**מיכל - מעצבת גרפית (28)**
- עובדת בסטודיו למיתוג
- מחפשת השראה לפרויקטים
- רוצה: מפות יפות, דוגמאות עיצוב, צבעים
- משתמש: Desktop + Tablet

**רון - מטייל תרמילאי (24)**
- תכנן טיול לאירופה
- צריך מידע מעשי על תחבורה
- רוצה: מידע בסיסי, מחירים, מפות פשוטות
- משתמש: Mobile בלבד

## 3. תכונות מרכזיות

### 3.1 תכונות ליבה (Phase 1)

✅ **מפת עולם אינטראקטיבית**
- סימון כל הערים עם מערכות תחבורה
- לחיצה על עיר = מעבר לדף המערכת
- Zoom, Pan, Clusters

✅ **מאגר מערכות**
- 20-30 ערים בשלב ראשון
- נתונים: אורך, תחנות, קווים, נוסעים, שנת פתיחה
- תמונות ומפות

✅ **חיפוש וסינון**
- חיפוש טקסט חופשי
- סינון לפי יבשת, סוג מערכת, גודל
- מיון לפי פרמטרים שונים

✅ **דפי מערכת מפורטים**
- מידע היסטורי
- קישור למפה רשמית
- פירוט קווים
- נתונים סטטיסטיים

✅ **PWA**
- התקנה למסך הבית
- עבודה במצב offline
- מהירות גבוהה

### 3.2 תכונות מתקדמות (Phase 2)

✅ **השוואת מערכות**
- בחירת 2-5 מערכות
- השוואה ויזואלית וסטטיסטית
- גרפים

✅ **ציר זמן היסטורי**
- כל האירועים מ-1863 עד היום
- סינון לפי אזורים
- אנימציה על מפת עולם

✅ **דף סטטיסטיקות**
- דשבורד עם גרפים
- Top 10 lists
- מפות חום
- שיאים

✅ **מפות SVG**
- יצירת מפות סכמטיות
- אינטראקטיביות
- צבעים נכונים

### 3.3 תכונות קהילה (Phase 3)

✅ **משתמשים**
- הרשמה והתחברות
- פרופיל אישי
- שמירת מועדפים

✅ **תוכן משתמשים**
- תגובות ודירוגים
- העלאת תמונות
- יומן מסעות

✅ **פורום**
- דיונים על מערכות
- שיתוף חוויות
- שאלות ותשובות

### 11.3 Service Worker - אסטרטגיות Caching

**קובץ: public/service-worker.js**

```javascript
// גרסה
const CACHE_VERSION = 'v1.0.0';
const CACHE_NAME = `transit-maps-${CACHE_VERSION}`;

// קבצים לcache מראש (App Shell)
const PRECACHE_URLS = [
  '/',
  '/index.html',
  '/manifest.json',
  '/styles/main.css',
  '/scripts/main.js',
  '/offline.html',
  // פונטים
  '/fonts/Assistant-Regular.woff2',
  '/fonts/Assistant-Bold.woff2',
  // אייקונים
  '/icons/icon-192x192.png',
  // תמונות חיוניות
  '/images/logo.svg',
  '/images/world-map-base.jpg'
];

// מערכות פופולריות ל-precache (20 ערים)
const POPULAR_SYSTEMS = [
  '/data/systems/london-underground.json',
  '/data/systems/paris-metro.json',
  '/data/systems/new-york-subway.json',
  '/data/systems/tokyo-metro.json',
  '/data/systems/moscow-metro.json',
  '/data/systems/shanghai-metro.json',
  '/data/systems/beijing-subway.json',
  '/data/systems/seoul-metro.json',
  '/data/systems/hong-kong-mtr.json',
  '/data/systems/madrid-metro.json',
  '/data/systems/barcelona-metro.json',
  '/data/systems/berlin-ubahn.json',
  '/data/systems/singapore-mrt.json',
  '/data/systems/toronto-ttc.json',
  '/data/systems/sydney-metro.json',
  '/data/systems/rome-metro.json',
  '/data/systems/vienna-ubahn.json',
  '/data/systems/athens-metro.json',
  '/data/systems/lisbon-metro.json',
  '/data/systems/amsterdam-metro.json'
];

// התקנה - cache קבצים בסיסיים
self.addEventListener('install', (event) => {
  console.log('[SW] Installing...');
  
  event.waitUntil(
    caches.open(CACHE_NAME).then((cache) => {
      console.log('[SW] Caching app shell and popular systems');
      return cache.addAll([...PRECACHE_URLS, ...POPULAR_SYSTEMS]);
    })
  );
  
  // Force the waiting service worker to become the active service worker
  self.skipWaiting();
});

// הפעלה - מחיקת cache ישן
self.addEventListener('activate', (event) => {
  console.log('[SW] Activating...');
  
  event.waitUntil(
    caches.keys().then((cacheNames) => {
      return Promise.all(
        cacheNames
          .filter((name) => name !== CACHE_NAME)
          .map((name) => {
            console.log('[SW] Deleting old cache:', name);
            return caches.delete(name);
          })
      );
    })
  );
  
  // Take control of all pages
  self.clients.claim();
});

// Fetch - אסטרטגיות שונות
self.addEventListener('fetch', (event) => {
  const { request } = event;
  const url = new URL(request.url);
  
  // Skip cross-origin requests
  if (url.origin !== location.origin) {
    return;
  }

  // 1. Network First - לנתונים דינמיים (API)
  if (url.pathname.startsWith('/api/')) {
    event.respondWith(networkFirst(request));
  }
  
  // 2. Cache First - לקבצים סטטיים
  else if (
    url.pathname.match(/\.(js|css|woff2|png|jpg|jpeg|svg|gif|webp)$/)
  ) {
    event.respondWith(cacheFirst(request));
  }
  
  // 3. Stale While Revalidate - למפות ונתונים
  else if (url.pathname.startsWith('/data/')) {
    event.respondWith(staleWhileRevalidate(request));
  }
  
  // 4. Network Only with offline fallback - לדפים
  else {
    event.respondWith(
      fetch(request).catch(() => {
        return caches.match('/offline.html');
      })
    );
  }
});

// אסטרטגיית Network First
async function networkFirst(request) {
  try {
    const response = await fetch(request);
    const cache = await caches.open(CACHE_NAME);
    cache.put(request, response.clone());
    return response;
  } catch (error) {
    const cached = await caches.match(request);
    return cached || new Response('Offline', { status: 503 });
  }
}

// אסטרטגיית Cache First
async function cacheFirst(request) {
  const cached = await caches.match(request);
  if (cached) {
    return cached;
  }
  
  try {
    const response = await fetch(request);
    const cache = await caches.open(CACHE_NAME);
    cache.put(request, response.clone());
    return response;
  } catch (error) {
    return new Response('Network error', { status: 408 });
  }
}

// אסטרטגיית Stale While Revalidate
async function staleWhileRevalidate(request) {
  const cached = await caches.match(request);
  
  const fetchPromise = fetch(request).then((response) => {
    const cache = caches.open(CACHE_NAME);
    cache.then((c) => c.put(request, response.clone()));
    return response;
  });
  
  return cached || fetchPromise;
}
```

```javascript
// src/components/InstallPrompt.jsx

import { useState, useEffect } from 'react';

export function InstallPrompt() {
  const [deferredPrompt, setDeferredPrompt] = useState(null);
  const [showInstall, setShowInstall] = useState(false);

  useEffect(() => {
    const handler = (e) => {
      e.preventDefault();
      setDeferredPrompt(e);
      setShowInstall(true);
    };

    window.addEventListener('beforeinstallprompt', handler);

    return () => {
      window.removeEventListener('beforeinstallprompt', handler);
    };
  }, []);

  const handleInstall = async () => {
    if (!deferredPrompt) return;

    deferredPrompt.prompt();
    const { outcome } = await deferredPrompt.userChoice;

    if (outcome === 'accepted') {
      console.log('✅ User accepted install');
    }

    setDeferredPrompt(null);
    setShowInstall(false);
  };

  const handleDismiss = () => {
    setShowInstall(false);
    localStorage.setItem('install-dismissed', Date.now());
  };

  if (!showInstall) return null;

  return (
    <div className="install-banner">
      <div className="banner-content">
        <img src="/icons/icon-72x72.png" alt="לוגו" />
        <div className="banner-text">
          <h3>התקן את "מפות תחבורה"</h3>
          <p>גישה מהירה, עבודה במצב לא מקוון</p>
        </div>
        <button onClick={handleInstall} className="btn-primary">
          התקן
        </button>
        <button onClick={handleDismiss} className="btn-text">
          ✕
        </button>
      </div>
    </div>
  );
}
```

```javascript
// src/hooks/useOfflineStatus.js

import { useState, useEffect } from 'react';

export function useOfflineStatus() {
  const [isOffline, setIsOffline] = useState(!navigator.onLine);

  useEffect(() => {
    const handleOnline = () => {
      setIsOffline(false);
      showOnlineStatus();
    };

    const handleOffline = () => {
      setIsOffline(true);
      showOfflineStatus();
    };

    window.addEventListener('online', handleOnline);
    window.addEventListener('offline', handleOffline);

    return () => {
      window.removeEventListener('online', handleOnline);
      window.removeEventListener('offline', handleOffline);
    };
  }, []);

  return isOffline;
}
```

```javascript
class DataService {
  constructor() {
    this.dataSource = 'json'; // 'json' | 'api' | 'database'
    this.cache = new Map();
  }

  // ממשק אחיד לטעינת מערכת
  async getSystem(id) {
    // בדיקת cache
    if (this.cache.has(id)) {
      return this.cache.get(id);
    }

    let data;
    
    switch (this.dataSource) {
      case 'json':
        data = await this.loadFromJSON(id);
        break;
      case 'api':
        data = await this.loadFromAPI(id);
        break;
      case 'database':
        data = await this.loadFromDB(id);
        break;
      default:
        throw new Error(`Unknown data source: ${this.dataSource}`);
    }

    // שמירה ב-cache
    this.cache.set(id, data);
    return data;
  }

  // טעינה מ-JSON
  async loadFromJSON(id) {
    const response = await fetch(`/data/systems/${id}.json`);
    if (!response.ok) {
      throw new Error(`System not found: ${id}`);
    }
    return response.json();
  }

  // טעינה מ-API
  async loadFromAPI(id) {
    const response = await fetch(`/api/systems/${id}`);
    if (!response.ok) {
      throw new Error(`Failed to fetch system: ${id}`);
    }
    return response.json();
  }

  // טעינה מ-Database (עתידי)
  async loadFromDB(id) {
    // TODO: implement database access
    throw new Error('Database access not implemented');
  }

  // טעינת כל המערכות
  async getAllSystems() {
    const response = await fetch('/data/systems-list.json');
    return response.json();
  }

  // חיפוש
  async searchSystems(query) {
    const allSystems = await this.getAllSystems();
    const lowerQuery = query.toLowerCase();
    
    return allSystems.filter(system => 
      system.name.he.toLowerCase().includes(lowerQuery) ||
      system.name.en.toLowerCase().includes(lowerQuery) ||
      system.city.he.toLowerCase().includes(lowerQuery) ||
      system.country.he.toLowerCase().includes(lowerQuery)
    );
  }

  // סינון
  async filterSystems(filters) {
    const allSystems = await this.getAllSystems();
    
    return allSystems.filter(system => {
      if (filters.continent && system.continent !== filters.continent) {
        return false;
      }
      if (filters.type && system.type !== filters.type) {
        return false;
      }
      if (filters.minStations && system.stats.stations < filters.minStations) {
        return false;
      }
      // ... עוד פילטרים
      return true;
    });
  }
}

export const dataService = new DataService();
```
