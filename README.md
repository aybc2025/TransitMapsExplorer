# 🚇 מפות תחבורה מכל העולם - Transit Maps Explorer

אתר PWA בעברית המציג מערכות תחבורה המונית מכל העולם - מטרו, רכבת קלה, חשמליות ו-BRT.

## ✨ תכונות Phase 1 (MVP)

- 🏠 **דף בית** - Hero section, מפת עולם אינטראקטיבית, מערכות מיוחדות
- 🗂️ **דף סקירה** - חיפוש, סינון לפי יבשת/סוג/סטטוס, מיון, תצוגת Grid/List/Map
- 📄 **דף מערכת** - נתונים מפורטים, קווים, מפות, עובדות מעניינות
- ℹ️ **דף אודות** - מידע על הפרויקט ומקורות
- 🌍 **25 מערכות תחבורה** - לונדון, פריז, ניו יורק, טוקיו ועוד
- 🇮🇱 **עברית מלאה** - RTL, פונטים עבריים

## 🛠️ טכנולוגיות

- React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Leaflet - מפות
- Framer Motion - אנימציות
- Zustand - state
- React Router v7
- Vite

## 🚀 התקנה

```bash
pnpm install
pnpm dev
```
